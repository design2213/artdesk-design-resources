# WordPress企业官网SEO/GEO Checklist

## SEO基础

- 每个页面有唯一SEO标题
- 每个页面有唯一Meta Description
- 每个核心页面只有一个明确H1
- H2/H3结构清楚
- sitemap可访问
- robots.txt允许抓取
- canonical设置正确
- 图片包含准确Alt文本

## GEO内容结构

- 首段直接回答页面主题
- 服务定义清楚
- 适用对象清楚
- 服务流程清楚
- 交付物清楚
- FAQ为页面可见内容
- 案例和服务页有内部链接
- 品牌主体信息一致

## Schema

- 首页包含Organization和WebSite
- 服务页包含Service
- FAQ页面或FAQ区块包含FAQPage
- 文章页包含Article
- 面包屑包含BreadcrumbList
- Schema内容与页面可见内容一致

## 性能

- 图片已压缩
- 首屏不加载过多大图
- 案例图启用懒加载
- 缓存插件配置正确
- 不重复安装同类插件
- 移动端首屏可读

## AI入口

- llms.txt可访问
- llms.txt包含核心服务页
- llms.txt包含FAQ和案例页
- 公开外部平台资料与官网品牌信息一致
